# 🎮 MTG MODS Bot

Telegram бот для приёма заказов на скрипты.

## 📦 Установка

### 1. Установи Node.js
Скачай с https://nodejs.org/

### 2. Открой терминал в папке с ботом
```bash
cd c:\Users\Ankon\Desktop\сайт
```

### 3. Запусти бота
```bash
node simple-bot.js
```

## 🤖 Как работает бот

**Пользователь пишет боту:**
1. `/start` — приветствие и меню
2. Отправляет описание заказа
3. Прикрепляет файлы (если есть)

**Бот пересылает заказ админу (тебе):**
- Все сообщения пересылаются в твой чат (ID: 5017662184)
- Ты видишь кто отправил и можешь ответить

## ⚙️ Настройка

## 🌐 Хостинг (чтобы бот работал 24/7)

### Вариант 1: Heroku
1. Зарегистрируйся на https://heroku.com
2. Создай новое приложение
3. Подключи GitHub с этим репозиторием
4. Добавь Procfile: `worker: node simple-bot.js`

### Вариант 2: Railway
1. https://railway.app
2. New Project → Deploy from GitHub
3. Бот запустится автоматически

### Вариант 3: VPS
- Арендуй сервер (Timeweb, Selectel, DigitalOcean)
- Установи Node.js
- Запусти через PM2: `pm2 start simple-bot.js`
