# 🎮 MTG MODS Bot

Telegram бот для приёма заказов на скрипты.

⚙️ Как это работает
Отправка: Вы оформляете заявку на сайте, данные автоматически уходят в бота.
Проверка: Заказ проходит модерацию.
Связь: О результате вам сообщат в личные сообщения (в тот чат/бот, где вы указали свой юзернейм).
📬 Статусы заявки
❌ Отказ: Вы получите уведомление в ЛС с причиной отказа.
✅ Принят: Вы получите уведомление в ЛС с реквизитами для оплаты.
